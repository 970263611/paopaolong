# 泡泡龙游戏 (Bubble Dragon Game)

这是一个使用 Java Swing 开发的经典泡泡龙游戏，支持双人对战模式。

## 🎮 游戏特性

- 🎯 经典泡泡龙玩法 + 双人对战模式
- 🎨 5 种不同颜色的泡泡
- 💫 连击消除系统和悬浮掉落机制
- 🏆 分数统计和胜负判定
- 🎵 程序生成的音效系统
- ⚙️ 可自定义的按键配置

## 🚀 快速开始

### 方式一：使用运行脚本（推荐）

**macOS / Linux：**
```bash
chmod +x run.sh
./run.sh
```

**Windows：**
双击 `run.bat` 或在命令提示符执行：
```cmd
run.bat
```

### 方式二：手动编译运行

```bash
# 创建输出目录
mkdir -p out

# 编译所有 Java 文件
javac -d out src/main/java/com/ppl/*.java

# 运行游戏
java -cp out com.ppl.BubbleDragonGame
```

### 方式三：使用 Maven

```bash
mvn clean compile exec:java -Dexec.mainClass="com.ppl.BubbleDragonGame"
```

## 🎯 游戏操作

### 玩家 1（左侧区域）
- **A 键** - 向左旋转
- **D 键** - 向右旋转
- **W 键** - 发射泡泡

### 玩家 2（右侧区域）
- **J 键** - 向左旋转
- **L 键** - 向右旋转
- **I 键** - 发射泡泡

### 通用操作
- **F1 或 P 键** - 打开按键设置对话框
- **鼠标移动** - 控制发射方向（单人模式）
- **鼠标点击** - 发射泡泡（单人模式）

💡 **提示**：按 F1/P 可以在游戏中修改按键配置（仅本次游戏有效）

## 📖 游戏规则

### 基本规则
1. **目标**：消除所有泡泡或将对方区域填满
2. **消除规则**：3 个或更多相同颜色的泡泡相连即可消除
3. **游戏结束**：当泡泡堆积到屏幕底部红色警戒线时失败
4. **胜利条件**：先将对方区域填满的玩家获胜

### 得分系统
- **基础消除**：每个泡泡 10 分
- **连击奖励**：超过 3 个的部分，每个 20 分
  - 3 个泡泡 = 30 分
  - 4 个泡泡 = 50 分
  - 5 个泡泡 = 70 分
- **掉落奖励**：每个因失去连接而掉落的泡泡 50 分

### 高级技巧
1. **预判落点**：观察泡泡飞行轨迹，提前规划
2. **连击策略**：制造大面积同色区域，触发连锁反应
3. **防守策略**：优先消除低处泡泡，防止形成"天花板"
4. **利用掉落**：让悬浮泡泡掉落可获得额外 50 分奖励

## 🔧 技术实现

### 核心技术栈
- **开发语言**：Java 26
- **GUI 框架**：Java Swing (Graphics2D)
- **构建工具**：Maven
- **游戏帧率**：60 FPS

### 核心算法
1. **六边形网格系统**
   - 奇数行偏移布局，模拟真实泡泡龙的蜂窝状结构
   - 每个泡泡最多 6 个邻居（上下左右 + 斜向）

2. **碰撞检测**
   - 圆形碰撞检测算法
   - 实时网格定位

3. **搜索算法**
   - **DFS（深度优先搜索）**：用于匹配相同颜色的泡泡
   - **BFS（广度优先搜索）**：用于检测悬浮泡泡

4. **物理模拟**
   - 重力作用：失去支撑的泡泡自动掉落
   - 边界处理：完善的越界保护

### 图形渲染
- 双缓冲渲染避免闪烁
- 抗锯齿处理平滑边缘
- 径向渐变绘制立体泡泡
- 半透明瞄准线

## ⚙️ 配置参数

### 游戏平衡参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| BUBBLE_RADIUS | 20px | 泡泡半径 |
| SHOOT_SPEED | 15 | 发射速度 |
| GRID_ROWS | 16 | 网格行数 |
| GRID_COLS | 15 | 网格列数 |
| DANGER_LINE | 第 15 行 | 游戏结束警戒线 |

### 音效系统
- **发射音效**：800Hz→200Hz 滑音效果
- **消除音效**：500Hz→400Hz→300Hz 三音调序列
- **程序生成**：无需外部文件，实时合成声波

## 🐛 常见问题

### 编译错误
**问题**：提示找不到符号或语法错误
**解决**：确保安装了 JDK 26 或更高版本
```bash
java -version
javac -version
```

### 游戏无法启动
1. 检查 Java 是否正确安装
2. 确认没有防火墙阻止
3. 尝试重新编译项目：`rm -rf out && mkdir out && javac -d out src/main/java/com/ppl/*.java`

### 按键不响应
1. 确保游戏窗口处于焦点状态
2. 按 F1/P 检查当前按键配置
3. 在按键设置中修改为其他按键测试

## 🎉 开发者提示

如果你想修改游戏参数，可以调整以下常量：

**Bubble.java：**
- `RADIUS`: 泡泡半径（默认 20）

**Player.java：**
- `shootSpeed`: 发射速度（默认 15）

**GamePanel.java：**
- `GRID_ROWS`: 网格行数（默认 16）
- `GRID_COLS`: 网格列数（默认 15）
- `dangerLine`: 游戏结束线位置（默认 GRID_ROWS - 1）

**KeyBindingsConfig.java：**
- `SINGLE_LEFT_KEY`: 玩家 1 左旋转键（默认 A=65）
- `SINGLE_RIGHT_KEY`: 玩家 1 右旋转键（默认 D=68）
- `SINGLE_SHOOT_KEY`: 玩家 1 发射键（默认 W=87）
- `P2_LEFT_KEY`: 玩家 2 左旋转键（默认 J=74）
- `P2_RIGHT_KEY`: 玩家 2 右旋转键（默认 L=76）
- `P2_SHOOT_KEY`: 玩家 2 发射键（默认 I=73）
